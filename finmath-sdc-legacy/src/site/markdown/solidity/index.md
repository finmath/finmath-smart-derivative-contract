#### Unit Tests

To run the unit tests, switch to directory `solidity`. Launch test nodes via hardhat (`npx hardhat node`).
Then run the unit test via hardhat (`npx hardhat test`).

```shell
cd solidity
npm i
npx hardhat node
npx hardhat test
```
